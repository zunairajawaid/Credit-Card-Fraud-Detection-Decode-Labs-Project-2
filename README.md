# 💳 Credit Card Fraud Detection - DecodeLabs Project 2

## 📌 Project Overview
This project focuses on detecting fraudulent credit card transactions using data analysis and machine learning techniques. The goal is to identify suspicious transactions accurately while minimizing false positives.

## 🎯 Objectives
- Explore and understand the credit card transaction dataset.
- Perform data preprocessing and cleaning.
- Handle missing values (if any).
- Analyze class imbalance.
- Train a machine learning model for fraud detection.
- Evaluate the model using appropriate performance metrics.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / VS Code

## 📂 Project Structure
```

```

## 📊 Workflow
1. Import the dataset.
2. Perform Exploratory Data Analysis (EDA).
3. Clean and preprocess the data.
4. Split the dataset into training and testing sets.
5. Train the machine learning model.
6. Evaluate the model.
7. Predict fraudulent transactions.

## 📈 Model Evaluation
The model is evaluated using:
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## 🚀 Results
The model successfully identifies fraudulent transactions using machine learning techniques and provides insights into transaction patterns.
